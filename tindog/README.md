# Tindog

Website for an imaginary tech startup. The site advertises a new app called Tindog, a dating app for dogs.

## Technologies

- Bootstrap 4.5
  - Button
  - Carousel
  - Card
  - Grid system
  - Pricing
- CSS 3
- Fontawesome
- Google Fonts
- HTML 5

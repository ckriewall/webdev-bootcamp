CSS

Element Selector

The element selector selects HTML elements based on the element name.

Syntax

selector { property : value }

Example

body { background-color : blue }

Implementation

<body>Body Content</body>

Class Selector

The class selector selects HTML elements with a specific class attribute.

Syntax

.selector { property : value }

Example

.bacon { background-color : blue }

Implementation

<body class="bacon">Body Content</body>

ID Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element.

Syntax

#selector { property : value }

Example

#bacon { background-color : blue }

Implementation

<body id="bacon">Body Content</body>
